Este repositorio contiene los ejemplos y ejercicios de la clase "Análisis contrafactual en demografía y los estudios de población". Para ello, se tiene dos archivos con los datos a utilizar en los ejercicios y su respectivo código de RMarkdown. El archivo en formato PDF, contiene la compilación del documento RMarkdown, se recomienda abrir ese archivo para la clase y posteriormente cada alumnos puede trabajar con el código por su cuenta.
